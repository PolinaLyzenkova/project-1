# Custom Game Mode Prompt

## Overview
Create a new version of the Cashflow game with custom rules and mechanics.

## Custom Mode Features

### Modified Rules
- Faster gameplay with reduced starting expenses
- Accelerated passive income growth
- Custom starting cash amounts
- Modified profession stats
- Different win conditions

### Game Mechanics Changes
- Starting cash multiplier: 1.5x
- Passive income multiplier: 1.2x
- Reduced base expenses: 0.8x
- Faster progression through Rat Race

### Visual Differences
- Different color scheme or theme
- Custom UI indicators for custom mode
- Modified board layout or spacing

## Implementation Requirements
- Should reuse core game logic where possible
- Maintain separate game state from standard mode
- Allow switching between standard and custom modes
- Preserve game functionality and balance
